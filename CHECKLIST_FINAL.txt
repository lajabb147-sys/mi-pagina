1. backend/server.js
   → El servidor principal de Node.js
   → Puerto 5000
   → Conectado a MySQL

2. backend/.env
   → Configuración de credenciales
   → Edita: DB_PASSWORD=tu_contraseña

3. backend/database.sql
   → Script para crear la base de datos
   → Ejecutar en MySQL Workbench

4. backend/index.html
   → Página web con HTML, CSS y JavaScript
   → Sin dependencias externas
   → Conecta con el backend API

5. backend/config/conexion.js
   → Conexión a la base de datos MySQL
   → Manejo de pool de conexiones

GET    /api/test              → Verificar servidor
GET    /api/cursos            → Ver todos los cursos
GET    /api/cursos/:id        → Ver un curso
POST   /api/cursos            → Crear curso
PUT    /api/cursos/:id        → Actualizar curso
DELETE /api/cursos/:id        → Eliminar curso
GET    /api/usuarios          → Ver usuarios
