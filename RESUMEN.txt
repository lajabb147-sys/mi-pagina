🚀 PASOS SIGUIENTES:

1️⃣  INSTALAR MySQL
   → Descarga desde: https://www.mysql.com/downloads/

2️⃣  INSTALAR Node.js
   → Descarga desde: https://nodejs.org/

3️⃣  CREAR LA BASE DE DATOS
   → Copia el contenido de "backend/database.sql"
   → Pega en MySQL Workbench y ejecuta

4️⃣  CONFIGURAR CREDENCIALES
   → Abre "backend/.env"
   → Completa con tus credenciales de MySQL:
      DB_USER=root
      DB_PASSWORD=tu_contraseña

5️⃣  INSTALAR DEPENDENCIAS
   → Terminal en "backend/"
   → npm install

6️⃣  INICIAR SERVIDOR
   → npm run dev
   → Verás: "Servidor corriendo en puerto 5000"

7️⃣  PROBAR
   → Abre: http://localhost:5000/api/test
   → O abre: backend/ejemplo-frontend.html
